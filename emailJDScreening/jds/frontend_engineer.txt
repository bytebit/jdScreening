# 前端开发工程师

## 岗位职责
- 负责公司 Web 产品的前端架构设计与开发
- 与后端工程师协作完成产品功能迭代
- 优化页面性能，提升用户体验

## 硬性要求（必需满足）
- 统招本科及以上学历，计算机相关专业
- 3 年以上前端开发经验
- 精通 HTML5 / CSS3 / JavaScript (ES6+)
- 熟练使用 Vue.js 或 React 框架
- 熟悉前端工程化（Webpack / Vite）

## 加分项
- 有 TypeScript 项目经验
- 有移动端 H5 开发经验
- 有大型项目性能优化经验
- 有开源项目贡献

## 工作地点
北京 / 上海 / 深圳
